DOM Yöntemleri

Metod Adı Açıklama Örnek Kullanım
getElementById(id) Belirtilen id özniteliğine sahip HTML öğesini döndürür. document.getElementById("myId")
getElementsByClassName(class) Belirtilen sınıf adına sahip tüm HTML öğelerini bir dizi olarak döndürür. document.getElementsByClassName("myClass")
getElementsByTagName(tag) Belirtilen etikete sahip tüm HTML öğelerini bir dizi olarak döndürür. document.getElementsByTagName("p")
querySelector(selector) Belirtilen CSS seçicisine göre ilk eşleşen öğeyi döndürür. document.querySelector("#myId .myClass")
querySelectorAll(selector) Belirtilen CSS seçicisine göre tüm eşleşen öğeleri bir NodeList nesnesi olarak döndürür. document.querySelectorAll("p.myClass")
createElement(tagName) Belirtilen etiket adıyla yeni bir HTML öğesi oluşturur. document.createElement("div")
appendChild(node) Bir düğümü belirtilen öğenin sonuna ekler. document.getElementById("myId").appendChild(newElement)
